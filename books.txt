Рекомендую мою книгу «Программирование. Принципы и практика с использованием C++» 
для тех, кто только начинает изучать программирование, и «A Tour of C++» для тех, 
кто уже программирует и хочет больше узнать о современном С++. Люди с хорошей 
математической базой могут начать с книги Питера Готтшлинга «Современный C++. 
Для программистов, инженеров и ученых».
Когда вы начнёте использовать C++ по-настоящему, вам понадобится набор правил, 
чтобы научиться лучшим практикам. Для этого на гитхабе есть C++ Core Guidelines.
